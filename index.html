AETHER NLP 시스템 안내

본 앱은 한글 학술 연구 및 정밀 텍스트 전처리를 클라이언트 환경에서 가공하여 고품질 가중치 분석을 수행할 수 있게 구축된 디지털 마이닝 대시보드입니다.

1. 데이터 업로드: PDF 또는 텍스트 파일을 드래그 앤 드롭하면, 실시간으로 한글 본문을 추출합니다. URL을 기입할 경우 지능형 본문 텍스트 자동 크롤러가 동작합니다.

2. 분석 엔진: 가중치 분석, LDA 모델, 시계열 동태 분석, 의미 네트워크 등을 동시 수행하며 원시 텍스트의 크기와 형태소 성향을 자동 측정합니다.

3. 내보내기 지원: 분석이 완료되면 논문이나 저널에 즉시 편집 제출할 수 있도록 정식 수치 보고서를 PDF 전용 출력 및 Word(.docx) 파일 포맷으로 출력해줍니다.
